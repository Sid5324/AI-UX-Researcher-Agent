### Interaction Specifications

**Auto-fill Behavior:**
1. **On Load:** Check for browser auto-fill data
2. **If Available:** Populate fields, show "Auto-filled from browser" badge
3. **Validation:** Run validation after 500ms delay
4. **Focus:** Move focus to first empty required field

**Real-time Validation:**
- **Email:** Validate format on blur (not while typing)
- **Address:** Validate via Google Places API on blur
- **ZIP:** Validate against USPS database (async)

**Navigation:**
- **Back Button:** Save form state to sessionStorage, navigate to cart
- **Continue:** Validate all fields → POST to /api/checkout/shipping → Navigate to payment on success
