## EXECUTIVE SUMMARY

### What This Is
An **autonomous AI agent** that pursues product research goals end-to-end without step-by-step human direction. Given a high-level objective like "fix our declining activation rate," the agent independently gathers data, forms hypotheses, runs experiments, validates solutions, and delivers implementation-ready recommendations—checking in with humans only at critical decision points.

### Core Philosophy Shift

| Traditional Tool (Old PRD) | Agentic AI (This PRD) |
|----------------------------|------------------------|
| User directs every step | Agent pursues goal autonomously |
| Sequential pipeline | Dynamic goal-seeking with loops |
| Fixed scenarios | Agent discovers what needs doing |
| Waits for input | Proactively seeks information |
| One-shot outputs | Iterative refinement until goal met |
| "What do you want me to do next?" | "I'm working on it, here's progress" |

### One-Line Value Proposition
**"Give me a goal, I'll solve it autonomously—you just approve at checkpoints."**

## 1. PRODUCT VISION

### The Problem with Non-Agentic Tools
Current UX research tools (including GPT, competitors) are **reactive assistants**:
- User: "Research competitors" → Tool: "Which ones?" → User: "Here are URLs" → Tool: "Here's data" → User: "Now analyze it"
- Requires constant hand-holding
- No memory between steps
- Can't recover from failures
- Doesn't learn from experience

### Our Solution: True Agentic AI
```
User: "Fix our declining activation rate. 2 weeks, $2k budget."

Agent: [Autonomously executes over 3 days]:
  ✓ Pulled Mixpanel data → Found 60% drop at "connect integration" step
  ✓ Analyzed 300 support tickets → "confused by OAuth" complaints ↑400%
  ✓ Recruited 8 users who dropped off → Conducted interviews
  ✓ Root cause: Users fear privacy violation
  ✓ Generated 3 solution hypotheses
  ✓ Tested all 3 with 20 users each
  ✓ Winner: Progressive permission (85% activation vs. 25% current)
  
  [CHECKPOINT] Approve this solution?
  
User: "Approved"

Agent: [Continues autonomously]:
  ✓ Generated PRD with full research trail
  ✓ Created Figma mockups
  ✓ Wrote Jira tickets
  ✓ Notified #product-team on Slack
  ✓ Scheduled stakeholder review for Thursday
  
  [COMPLETE] Solution ready for engineering.
```

**User involvement:** 2 checkpoint approvals vs. 20+ manual steps

## 2. CORE AGENTIC CAPABILITIES

### The ReAct Loop (Reason + Act)

```
┌─────────────────────────────────────────┐
│           GOAL ACCEPTED                 │
│  "Fix activation rate decline"         │
└─────────────────────────────────────────┘
                   ↓
         ┌─────────────────┐
         │   THINK         │  ← Explicit reasoning
         │  "I need data   │     (Chain-of-Thought)
         │   on drop-off"  │
         └────────┬────────┘
                  ↓
         ┌─────────────────┐
         │   ACT           │  ← Execute tools
         │  Pull Mixpanel  │     (Real API calls)
         │  analytics      │
         └────────┬────────┘
                  ↓
         ┌─────────────────┐
         │   OBSERVE       │  ← Perceive results
         │  "60% drop at   │
         │   step 3"       │
         └────────┬────────┘
                  ↓
         ┌─────────────────┐
         │   LEARN         │  ← Update beliefs
         │  Hypothesis:    │     Store in memory
         │  "Step 3 issue" │
         └────────┬────────┘
                  ↓
              [LOOP BACK TO THINK]
              (Until goal achieved)
```

### Key Differentiators

**1. Autonomous Execution**
- Agent doesn't ask "what next?" at every step
- Operates independently for hours/days
- User sees live progress, intervenes only if needed

**2. Dynamic Planning**
- Generates conditional task graphs: "If X fails, try Y"
- Replans when obstacles encountered
- Parallel execution of independent tasks

**3. Self-Correction**
- Detects when actions fail
- Analyzes root cause (reasoning error vs. tool failure)
- Automatically tries alternative approaches

**4. Multi-Layer Memory**
- **Working memory:** Current goal state, active hypotheses
- **Episodic memory:** What happened this session
- **Semantic memory:** Knowledge from all past projects
- **Procedural memory:** Learned skills ("how to do X")

**5. Real Tool Execution**
- Not just "suggests using Mixpanel" → Actually calls Mixpanel API
- Not just "draft email" → Actually sends email (if approved)
- Not just "mockup description" → Actually generates visual file

**6. Proactive Communication**
- Agent: "Stuck on X, need your input"
- Agent: "Found critical issue, pausing for approval"
- Agent: "Completed sub-goal, moving to next"

class GoalParser:
    def parse(self, user_input: str) -> Mission:
        """
        User: "Users are abandoning checkout. Fix it. 2 weeks, $2k."
        
        Agent extracts:
        {
          "primary_goal": "Reduce checkout abandonment",
          "success_criteria": "Abandonment rate < 30% (currently 60%)",
          "constraints": {
            "time": "2 weeks",
            "budget": "$2,000",
            "no_disrupt": "Don't break existing checkout"
          },
          "sub_goals": [
            "Identify abandonment cause",
            "Validate with users",
            "Design solution",
            "Test solution",
            "Implement"
          ],
          "uncertainties": [
            "Which step has highest drop-off?",
            "Is it UX or pricing issue?",
            "Technical feasibility of fixes?"
          ],
          "stakeholders": ["PM", "Design", "Engineering"],
          "autonomy_level": "supervised"  # Default: check in at key points
        }
        """
        
        prompt = f"""
        Parse this product goal into a structured mission.
        
        User input: "{user_input}"
        
        Extract:
        - Primary measurable goal
        - Success criteria (specific numbers if possible)
        - Time and budget constraints
        - What we need to discover
        - Appropriate autonomy level (based on risk)
        
        Return JSON with mission structure.
        """
        
        return self.llm.generate_json(prompt, model='claude-sonnet-4')

**Dynamic Task Graph Generation:**

```python
class TaskPlanner:
    def plan(self, mission: Mission) -> TaskGraph:
        """
        Generates conditional execution plan
        
        Example output for "fix checkout abandonment":
        
        TASK_1: Pull analytics data [AUTO]
          ↓
        TASK_2: Identify drop-off point [AUTO]
          ↓
        DECISION_1: If clear pattern found
          ├─ YES → TASK_3A: Form hypothesis [AUTO]
          └─ NO  → TASK_3B: User interviews [ASK_USER]
                    ↓
        TASK_4: Design experiments (3 options) [AUTO]
          ↓
        CHECKPOINT: Get user approval for experiments [WAIT_USER]
          ↓
        TASK_5: Run experiments in parallel [AUTO]
          ├─ Experiment A (pricing change)
          ├─ Experiment B (UX simplification)
          └─ Experiment C (social proof)
          ↓
        TASK_6: Analyze results [AUTO]
          ↓
        DECISION_2: If winner found (95% confidence)
          ├─ YES → TASK_7: Generate implementation package [AUTO]
          └─ NO  → TASK_7B: Run additional validation [AUTO]
          
        CHECKPOINT: Final approval [WAIT_USER]
          ↓
        TASK_8: Execute implementation [AUTO]
          ├─ Create Jira tickets
          ├─ Notify stakeholders
          └─ Set up success monitoring
        """
        
        # LLM generates plan based on goal type
        plan_prompt = f"""
        Generate execution plan for this mission:
        {mission.to_dict()}
        
        Consider:
        - What data is needed first?
        - What are decision points?
        - What can run in parallel?
        - What requires human approval?
        - What are fallback paths if tasks fail?
        
        Return task graph with:
        - Dependencies (A must complete before B)
        - Conditional branches (if X then Y else Z)
        - Checkpoints (where human approval needed)
        - Parallel tasks (can run simultaneously)
        """
        
        return self.llm.generate_task_graph(plan_prompt)
    
    def replan(self, graph: TaskGraph, failed_task: Task, error: Error) -> TaskGraph:
        """
        Self-correction: when task fails, generate new plan
        """
        
        replan_prompt = f"""
        Task '{failed_task.name}' failed with error: {error.message}
        
        Current goal: {graph.mission.primary_goal}
        Completed tasks: {graph.completed_tasks}
        Remaining budget: {graph.remaining_budget}
        
        Options:
        1. Retry with different approach
        2. Skip task and adjust later tasks
        3. Abandon current path, pursue alternative
        4. Ask user for guidance
        
        What should we do? Generate revised plan.
        """
        
        revised = self.llm.generate_task_graph(replan_prompt)
        return revised
```

### 4.2 ReAct Engine (Core Loop)

**The Continuous Reasoning-Action Loop:**

```python
class ReActEngine:
    def __init__(self):
        self.working_memory = WorkingMemory()
        self.episodic_memory = EpisodicMemory()
        self.tool_registry = ToolRegistry()
        self.autonomy_manager = AutonomyManager()
    
    async def run_until_goal_met(self, mission: Mission):
        """
        Main agentic loop - runs autonomously until goal achieved
        """
        
        current_observation = None
        
        while not mission.is_complete():
            
            # 1. THINK: Reason about current state
            thought = await self.think(
                observation=current_observation,
                goal=mission.current_goal,
                memory=self.working_memory.context,
                history=self.episodic_memory.recent_events(n=10)
            )
            
            self.episodic_memory.record('thought', thought)
            
            # 2. ACT: Select and execute actions
            actions = await self.select_actions(
                thought=thought,
                available_tools=self.tool_registry.available(),
                constraints=mission.constraints
            )
            
            for action in actions:
                
                # Check if need permission
                if self.autonomy_manager.needs_approval(action, mission):
                    approval = await self.request_user_approval(action)
                    
                    if not approval.granted:
                        # User rejected, replan
                        alternative = await self.replan_alternative(
                            action, 
                            approval.feedback
                        )
                        if alternative:
                            action = alternative
                        else:
                            continue  # Skip this action
                
                # Execute action
                result = await self.execute_action(action)
                
                self.episodic_memory.record('action', action, result)
                
                # 3. OBSERVE: What happened?
                observation = await self.observe_result(result)
                
                self.episodic_memory.record('observation', observation)
                
                # 4. LEARN: Update beliefs and memory
                await self.update_beliefs(observation, thought, action)
                
                # Self-correction if failed
                if not result.success:
                    correction = await self.self_correct(action, result)
                    if correction:
                        # Try correction immediately
                        result = await self.execute_action(correction)
                        observation = await self.observe_result(result)
                
                current_observation = observation
            
            # Check if goal achieved
            if await self.goal_achieved(mission):
                await self.complete_mission(mission)
                break
            
            # Update working memory with latest state
            self.working_memory.update(thought, actions, observation)
    
    async def think(self, observation, goal, memory, history) -> Thought:
        """
        Explicit reasoning step - not just pattern matching
        """
        
        prompt = f"""
        You are an autonomous product research agent reasoning about your next step.
        
        CURRENT GOAL: {goal.description}
        PROGRESS: {goal.completion_percent}%
        LAST OBSERVATION: {observation}
        WORKING CONTEXT: {memory}
        RECENT HISTORY: {history}
        
        Think through:
        1. What does the last observation tell us?
        2. Does it confirm or challenge our hypothesis?
        3. What uncertainty remains?
        4. What's the most logical next action to advance the goal?
        5. What could go wrong?
        6. What's our backup if it fails?
        
        Generate structured reasoning:
        {{
          "interpretation": "what this observation means",
          "hypothesis_status": "confirmed | challenged | needs_more_data",
          "confidence_change": "+15% or -10%",
          "uncertainties": ["list what we don't know"],
          "next_action": "specific next step with reasoning",
          "risks": ["potential failure modes"],
          "contingency": "if fails, do X"
        }}
        """
        
        thought = await self.llm.generate_json(prompt, model='claude-sonnet-4')
        return Thought(thought)
    
    async def select_actions(self, thought: Thought, available_tools: List[Tool], 
                             constraints: Dict) -> List[Action]:
        """
        Choose which tools to use based on reasoning
        """
        
        prompt = f"""
        Based on this reasoning, which tools should be executed?
        
        REASONING: {thought.interpretation}
        NEXT ACTION: {thought.next_action}
        AVAILABLE TOOLS: {[t.name for t in available_tools]}
        CONSTRAINTS: Budget ${constraints.budget_remaining}, 
                     Time {constraints.time_remaining}
        
        Select 1-3 tools that advance the goal.
        For each, specify exact parameters.
        
        Return:
        {{
          "actions": [
            {{
              "tool": "tool_name",
              "reason": "why this advances the goal",
              "params": {{"key": "value"}},
              "estimated_cost": "$50",
              "estimated_time": "2 hours",
              "risk_level": "low | medium | high",
              "fallback_tool": "alternative if this fails"
            }}
          ]
        }}
        """
        
        selection = await self.llm.generate_json(prompt, model='claude-sonnet-4')
        
        actions = []
        for s in selection['actions']:
            tool = self.tool_registry.get(s['tool'])
            actions.append(Action(
                tool=tool,
                params=s['params'],
                reason=s['reason'],
                risk=s['risk_level'],
                fallback=s.get('fallback_tool')
            ))
        
        return actions
    
    async def self_correct(self, failed_action: Action, result: Result) -> Action:
        """
        Agent detects and corrects its own errors
        """
        
        prompt = f"""
        An action just failed. Analyze and correct.
        
        FAILED ACTION: {failed_action.tool.name}({failed_action.params})
        ERROR: {result.error.message}
        GOAL: {self.working_memory.current_goal}
        
        Diagnose:
        1. Was this a reasoning error? (We misunderstood the problem)
        2. Was this a tool failure? (Tool didn't work, try alternative)
        3. Was this a parameter error? (Wrong inputs)
        4. Is the goal itself wrong? (We're solving the wrong problem)
        
        Return:
        {{
          "error_type": "reasoning | tool_failure | parameters | goal_misalignment",
          "analysis": "what went wrong",
          "correction": {{
            "action": "what to do instead",
            "tool": "alternative tool if needed",
            "params": {{}} "corrected parameters"
          }}
        }}
        """
        
        diagnosis = await self.llm.generate_json(prompt, model='claude-sonnet-4')
        
        if diagnosis['error_type'] == 'tool_failure' and failed_action.fallback:
            # Try fallback tool
            fallback_tool = self.tool_registry.get(failed_action.fallback)
            return Action(
                tool=fallback_tool,
                params=failed_action.params,
                reason=f"Fallback after {failed_action.tool.name} failed"
            )
        
        elif diagnosis['correction']:
            # Try corrected action
            return Action(
                tool=self.tool_registry.get(diagnosis['correction']['tool']),
                params=diagnosis['correction']['params'],
                reason=diagnosis['analysis']
            )
        
        return None  # Can't recover, need to replan
```

    def execute(self, action: Action) -> Result:
        """
        Execute tool with error handling, retries, and cost tracking
        """
        
        tool = action.tool
        
        # Pre-execution checks
        if not tool.validate_params(action.params):
            return Result.error("Invalid parameters")
        
        if action.estimated_cost > self.remaining_budget:
            return Result.error("Exceeds budget")
        
        # Execute with monitoring
        try:
            start_time = time.time()
            
            result = tool.execute(action.params)
            
            execution_time = time.time() - start_time
            
            # Log execution
            self.log_execution(tool, action, result, execution_time)
            
            # Verify result quality
            if not tool.verify_result(result):
                return Result.error("Tool succeeded but output invalid")
            
            return Result.success(result)
            
        except ToolExecutionError as e:
            # Attempt automatic recovery
            if e.recoverable and action.retry_count < 3:
                action.retry_count += 1
                return self.execute(action)  # Retry
            
            return Result.error(str(e), recoverable=False)
        
        except Exception as e:
            return Result.error(f"Unexpected error: {str(e)}")

class MixpanelAnalytics(Tool):
    """Real Mixpanel API integration"""
    
    def execute(self, params: Dict) -> Dict:
        import requests
        
        response = requests.get(
            'https://mixpanel.com/api/2.0/funnels',
            auth=(MIXPANEL_API_KEY, ''),
            params={
                'funnel_id': params['funnel_id'],
                'from_date': params['start_date'],
                'to_date': params['end_date']
            }
        )
        
        data = response.json()
        
        return {
            'steps': data['data']['steps'],
            'overall_conversion': data['data']['analysis']['overall_conv_ratio'],
            'drop_off_points': self._identify_drop_offs(data)
        }
    
    def _identify_drop_offs(self, data):
        """Find steps with biggest drop"""
        steps = data['data']['steps']
        drop_offs = []
        
        for i in range(len(steps) - 1):
            drop_rate = 1 - (steps[i+1]['count'] / steps[i]['count'])
            if drop_rate > 0.3:  # >30% drop
                drop_offs.append({
                    'from_step': steps[i]['event'],
                    'to_step': steps[i+1]['event'],
                    'drop_rate': drop_rate
                })
        
        return drop_offs

class MemorySystem:
    def __init__(self):
        self.working = WorkingMemory()      # Current goal context
        self.episodic = EpisodicMemory()    # Session history
        self.semantic = SemanticMemory()    # Long-term knowledge
        self.procedural = ProceduralMemory() # Learned skills

class WorkingMemory:
    """
    Active context for current goal
    Short-term, volatile (cleared after goal completion)
    """
    
    def __init__(self):
        self.current_goal = None
        self.active_hypotheses = []
        self.observations = []
        self.constraints = {}
        self.progress = 0.0
    
    def update(self, thought: Thought, actions: List[Action], 
               observation: Observation):
        """Update after each ReAct cycle"""
        
        # Update hypotheses based on new evidence
        for hyp in self.active_hypotheses:
            if observation.confirms(hyp):
                hyp.confidence += 0.15
            elif observation.refutes(hyp):
                hyp.confidence -= 0.20
        
        # Track progress toward goal
        self.progress = self._calculate_progress()
        
        # Prune weak hypotheses
        self.active_hypotheses = [h for h in self.active_hypotheses 
                                   if h.confidence > 0.3]
    
    def context_for_llm(self) -> str:
        """Compress working memory for LLM context"""
        
        return f"""
        CURRENT GOAL: {self.current_goal.description}
        PROGRESS: {self.progress}%
        
        ACTIVE HYPOTHESES:
        {self._format_hypotheses()}
        
        KEY OBSERVATIONS:
        {self._format_observations(top_k=5)}
        
        CONSTRAINTS:
        {self.constraints}
        """

class EpisodicMemory:
    """
    What happened during this session
    Persists for duration of goal, then archived
    """
    
    def __init__(self):
        self.events = []
        self.session_start = datetime.now()
    
    def record(self, event_type: str, content: Any, **metadata):
        """Record every thought, action, observation"""
        
        event = {
            'timestamp': datetime.now(),
            'type': event_type,  # 'thought' | 'action' | 'observation' | 'user_input'
            'content': content,
            'metadata': metadata
        }
        
        self.events.append(event)
    
    def recent_events(self, n: int = 10) -> List[Dict]:
        """Get last N events for context"""
        return self.events[-n:]
    
    def get_relevant_to(self, query: str) -> List[Dict]:
        """Retrieve events relevant to current reasoning"""
        
        query_embedding = get_embedding(query)
        
        scored = []
        for event in self.events:
            event_text = str(event['content'])
            event_embedding = get_embedding(event_text)
            similarity = cosine_similarity(query_embedding, event_embedding)
            scored.append((similarity, event))
        
        scored.sort(reverse=True)
        return [event for _, event in scored[:5]]
    
    def summarize_for_checkpoint(self) -> str:
        """Human-readable summary of what agent did"""
        
        summary = []
        
        # Group by action type
        actions_taken = [e for e in self.events if e['type'] == 'action']
        key_observations = [e for e in self.events if e['type'] == 'observation' 
                           and e.get('metadata', {}).get('important')]
        
        summary.append(f"Actions taken: {len(actions_taken)}")
        for action in actions_taken[-10:]:  # Last 10 actions
            summary.append(f"  • {action['content'].tool.name}: {action['content'].reason}")
        
        summary.append(f"\nKey findings:")
        for obs in key_observations[-5:]:
            summary.append(f"  • {obs['content']}")
        
        return "\n".join(summary)

class SemanticMemory:
    """
    Long-term knowledge across ALL projects
    Persists forever, grows with experience
    """
    
    def __init__(self, vector_db: VectorDatabase):
        self.db = vector_db
        self.insights = []
    
    def store(self, insight: Insight):
        """Store validated learning"""
        
        # Generate embedding
        embedding = get_embedding(insight.to_text())
        
        # Save to vector DB
        self.db.insert(
            id=insight.id,
            embedding=embedding,
            metadata={
                'type': insight.type,  # 'validated_hypothesis' | 'pain_point' | 'user_behavior'
                'content': insight.content,
                'confidence': insight.confidence,
                'evidence': insight.evidence,
                'project_id': insight.project_id,
                'created': insight.created_at,
                'tags': insight.tags,
                'validation_method': insight.validation_method,
                'effect_size': insight.effect_size
            }
        )
    
    def retrieve_relevant(self, query: str, context: Dict, top_k: int = 5) -> List[Insight]:
        """Find past insights relevant to current situation"""
        
        # Vector search
        query_embedding = get_embedding(query)
        candidates = self.db.search(query_embedding, top_k=20)
        
        # Re-rank by contextual relevance
        scored = []
        for candidate in candidates:
            relevance = self._calculate_relevance(candidate, context)
            scored.append((relevance, candidate))
        
        scored.sort(reverse=True)
        return [Insight.from_dict(c) for _, c in scored[:top_k]]
    
    def _calculate_relevance(self, insight: Dict, context: Dict) -> float:
        """Multi-factor relevance scoring"""
        
        scores = {
            # Semantic similarity (from vector search)
            'embedding_match': insight.get('_score', 0.0),
            
            # Domain match
            'industry_match': 1.0 if insight['tags'].get('industry') == context.get('industry') else 0.3,
            
            # Stage match
            'stage_match': 1.0 if insight['tags'].get('stage') == context.get('stage') else 0.5,
            
            # Recency (newer = more relevant)
            'recency': self._recency_score(insight['created']),
            
            # Confidence (higher = more trustworthy)
            'confidence': insight['confidence'],
            
            # Usage count (proven useful)
            'proven': min(insight.get('times_used', 0) / 10, 1.0)
        }
        
        weights = {
            'embedding_match': 0.30,
            'industry_match': 0.20,
            'stage_match': 0.15,
            'recency': 0.15,
            'confidence': 0.15,
            'proven': 0.05
        }
        
        return sum(scores[k] * weights[k] for k in scores)
    
    def cross_project_patterns(self) -> List[Pattern]:
        """Find patterns that appear in multiple projects"""
        
        # Cluster insights by similarity
        clusters = self.db.cluster(min_size=3)
        
        patterns = []
        for cluster in clusters:
            if len(cluster) >= 3:  # Appears in 3+ projects
                pattern = Pattern(
                    theme=self._extract_theme(cluster),
                    occurrences=len(cluster),
                    confidence=sum(i['confidence'] for i in cluster) / len(cluster),
                    examples=cluster[:3]
                )
                patterns.append(pattern)
        
        return sorted(patterns, key=lambda p: p.confidence, reverse=True)

class ProceduralMemory:
    """
    How to do things - learned skills
    Improves with experience (reinforcement learning)
    """
    
    def __init__(self):
        self.skills = {}
    
    def learn_from_success(self, task_name: str, execution_trace: List[Action]):
        """Learn from successful execution"""
        
        # Generalize from specific instance
        pattern = self._generalize_trace(execution_trace)
        
        if task_name in self.skills:
            # Update existing skill
            skill = self.skills[task_name]
            skill['pattern'] = self._merge_patterns(skill['pattern'], pattern)
            skill['success_count'] += 1
            skill['success_rate'] = skill['success_count'] / skill['total_attempts']
        else:
            # Create new skill
            self.skills[task_name] = {
                'pattern': pattern,
                'success_count': 1,
                'total_attempts': 1,
                'success_rate': 1.0,
                'contexts': []
            }
    
    def learn_from_failure(self, task_name: str, execution_trace: List[Action], error: Error):
        """Learn what NOT to do"""
        
        if task_name in self.skills:
            skill = self.skills[task_name]
            skill['total_attempts'] += 1
            skill['success_rate'] = skill['success_count'] / skill['total_attempts']
            
            # Record failure pattern to avoid
            skill.setdefault('anti_patterns', []).append({
                'trace': execution_trace,
                'error': error.message
            })
    
    def apply_skill(self, task_name: str, current_context: Dict) -> List[Action]:
        """Apply learned skill to new situation"""
        
        skill = self.skills.get(task_name)
        if not skill:
            return None
        
        if skill['success_rate'] < 0.5:
            return None  # Don't use low-confidence skills
        
        # Adapt pattern to current context
        adapted_actions = self._adapt_pattern(skill['pattern'], current_context)
        
        return adapted_actions

class AutonomyManager:
    """
    Decides when agent needs human approval
    Configurable autonomy levels
    """
    
    LEVELS = {
        'none': {
            'description': 'Agent suggests, user approves every action',
            'auto_execute': False,
            'checkpoint_frequency': 'every_action'
        },
        'partial': {
            'description': 'Agent executes data gathering, asks for conclusions',
            'auto_execute': True,
            'checkpoint_frequency': 'before_conclusions',
            'auto_allowed': ['data_gathering', 'analysis']
        },
        'supervised': {
            'description': 'Agent executes most tasks, checks in at key decisions',
            'auto_execute': True,
            'checkpoint_frequency': 'critical_only',
            'auto_allowed': ['data_gathering', 'analysis', 'content_creation']
        },
        'full': {
            'description': 'Agent executes everything, user reviews final output',
            'auto_execute': True,
            'checkpoint_frequency': 'final_only',
            'manual_pause': True
        }
    }
    
    def __init__(self, default_level='supervised'):
        self.current_level = default_level
        self.budget_tracker = BudgetTracker()
        self.risk_assessor = RiskAssessor()
    
    def needs_approval(self, action: Action, mission: Mission) -> bool:
        """Decide if action requires human approval"""
        
        level = self.LEVELS[self.current_level]
        
        # Always ask for high-risk actions
        if action.risk_level == 'high':
            return True
        
        # Always ask if cost exceeds threshold
        if action.estimated_cost > mission.approval_threshold_cost:
            return True
        
        # Always ask for irreversible actions
        if action.irreversible:
            return True
        
        # Check autonomy level
        if not level['auto_execute']:
            return True  # 'none' level - ask for everything
        
        if level['checkpoint_frequency'] == 'every_action':
            return True
        
        if level['checkpoint_frequency'] == 'before_conclusions':
            return action.type in ['conclusion', 'recommendation', 'decision']
        
        if level['checkpoint_frequency'] == 'critical_only':
            return action.type in ['user_facing_change', 'external_communication', 
                                  'budget_commitment']
        
        if level['checkpoint_frequency'] == 'final_only':
            return mission.is_near_completion()
        
        return False
    
    async def request_approval(self, action: Action) -> Approval:
        """Present action to user for approval"""
        
        presentation = {
            'action': action.description,
            'reason': action.reason,
            'cost': action.estimated_cost,
            'time': action.estimated_time,
            'risk': action.risk_level,
            'alternatives': action.alternatives,
            'consequences': {
                'if_approved': action.expected_outcome,
                'if_rejected': action.rejection_impact
            }
        }
        
        # Send to UI, wait for user response
        response = await self.ui.show_approval_dialog(presentation)
        
        return Approval(
            granted=response.approved,
            feedback=response.feedback,
            modifications=response.modifications
        )

class BudgetTracker:
    """Track spending across time, money, API calls"""
    
    def __init__(self, limits: Dict):
        self.limits = limits
        self.spent = {
            'money': 0.0,
            'time_hours': 0.0,
            'api_calls': 0
        }
    
    def can_afford(self, action: Action) -> bool:
        """Check if action fits within budget"""
        
        projected = {
            'money': self.spent['money'] + action.estimated_cost,
            'time_hours': self.spent['time_hours'] + action.estimated_time_hours,
            'api_calls': self.spent['api_calls'] + action.api_calls
        }
        
        for resource, limit in self.limits.items():
            if projected[resource] > limit:
                return False
        
        return True
    
    def record_spending(self, action: Action, actual: Dict):
        """Update after action completes"""
        
        self.spent['money'] += actual.get('cost', action.estimated_cost)
        self.spent['time_hours'] += actual.get('time_hours', action.estimated_time_hours)
        self.spent['api_calls'] += actual.get('api_calls', action.api_calls)

class RiskAssessor:
    """Evaluate action risk levels"""
    
    def assess(self, action: Action, context: Dict) -> str:
        """Return 'low' | 'medium' | 'high'"""
        
        risk_factors = {
            'irreversibility': action.irreversible,
            'user_facing': action.affects_users,
            'cost': action.estimated_cost > 500,
            'external': action.involves_external_parties,
            'data_deletion': action.modifies_data and not action.has_backup
        }
        
        risk_count = sum(risk_factors.values())
        
        if risk_count == 0:
            return 'low'
        elif risk_count <= 2:
            return 'medium'
        else:
            return 'high'

## 10. RISKS & MITIGATIONS

| Risk | Impact | Probability | Mitigation |
|------|--------|-------------|------------|
| **Agent makes wrong decisions** | High | Medium | • Checkpoint system (human approval)<br>• Confidence scoring<br>• Audit log for review<br>• Kill switch |
| **LLM hallucinations** | High | Medium | • Show sources for every claim<br>• Validate tool outputs<br>• Cross-reference with semantic memory |
| **Tool failures cascade** | Medium | Medium | • Fallback tools for each action<br>• Self-correction loop<br>• Graceful degradation |
| **Budget overruns** | Medium | Low | • Real-time budget tracker<br>• Cost approval thresholds<br>• Hard caps on spending |
| **User loses trust after failure** | High | Low | • Transparent reasoning trail<br>• Agent admits uncertainty<br>• Easy override/correction |
| **Privacy/security issues** | High | Low | • No PII stored without consent<br>• Encrypted data at rest<br>• User controls data access |
| **Complex goals agent can't handle** | Medium | High | • Agent detects complexity<br>• Suggests breaking into sub-goals<br>• Asks for human guidance early |
| **Tool API changes break agent** | Medium | Medium | • Abstract tool interface<br>• Version pinning<br>• Automated testing<br>• Fallback to manual |
