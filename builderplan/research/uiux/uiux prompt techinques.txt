High-quality UI images usually include:

* Device framing (phone / browser window)
* Clear layout hierarchy
* Design system cues (cards, shadows, spacing)
* Realistic typography
* Neutral background
* Specific UI components
* Lighting / rendering style
* UX context (dashboard, fintech, analytics)
* Visual clarity keywords

Think like this formula:

```
Context + Screen Type + Components + Style + Camera + Quality
```
