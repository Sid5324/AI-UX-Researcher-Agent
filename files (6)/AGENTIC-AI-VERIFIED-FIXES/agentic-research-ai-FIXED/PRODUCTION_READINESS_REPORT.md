## End-to-End User Workflow
1. Register user: `POST /auth/register`
2. Login user: `POST /auth/login`
3. Create workspace: `POST /workspaces/`
4. Create autonomous research goal: `POST /goals`
5. Monitor progress: `GET /goals`, `GET /goals/{goal_id}`
6. Live updates: `WS /ws/{goal_id}`
7. Review output fields: `goal.findings`, `goal.final_output`, checkpoints and agent states
